## Dependências

As seguintes bibliotecas Python são necessárias para a execução do script:

* pandas
* numpy
* matplotlib
* scikit-learn

Para instalar todas as dependências, execute o comando abaixo no terminal:

- pip install pandas numpy matplotlib scikit-learn

## Como Executar

1. Certifique-se de que o arquivo main.py e o dataset AirQualityUCI.csv estejam na mesma pasta.
2. Abra o terminal ou prompt de comando no diretório do projeto.
3. Execute o script com o comando:

- python main.py
